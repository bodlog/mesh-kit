MeshKit is provides an cross platfrom iOS, tvOS, and macOS mesh networking infrastructure

## Installation

MeshKit works best with [Carthage](https://github.com/carthage/carthage).

```swift
github "joeblau/mesh-kit"
```


## Use

```swift
import MeshKit
```
